# Neighbly: Система Индекса Архитектурной Гармонии

**Аналитический движок для расчета Индекса Архитектурной Гармонии и генерации рекомендаций по улучшению жилых пространств.**

Этот проект представляет собой программную реализацию концепции **Индекса Архитектурной Гармонии** — метрики, которая количественно оценивает, насколько пространство соответствует идеалам функции и красоты.

## Философия проекта

Проект основан на идее, что архитектура — это синтез науки и искусства, функции и эстетики. Качественное пространство — это не просто набор стен и мебели, а сбалансированная система, которая влияет на наше самочувствие, продуктивность и эмоциональное состояние.

**Система Индекса Архитектурной Гармонии** — это попытка кодифицировать ключевые принципы архитектурной гармонии, превращая их из абстрактных концепций в измеримые показатели. Этот инструмент позволяет провести объективный аудит любой комнаты, выявить её сильные и слабые стороны и предложить научно обоснованные способы её улучшения.

## Что такое Индекс Архитектурной Гармонии?

Индекс Архитектурной Гармонии (от 0 до 1) — это комплексный показатель, состоящий из двух ключевых компонентов:

### 1. **Индекс Функции**

Он оценивает, насколько эффективно и комфортно пространство с физической и эргономической точек зрения. Он рассчитывается на основе двух модулей:

  * **Термодинамический анализ:** Оценивает теплопотери помещения на основе строительных материалов, площади остекления и климатических условий.
  * **Оптический анализ:** Оценивает качество естественного освещения, учитывая размер и ориентацию окон, географическое положение и коэффициенты отражения поверхностей.

### 2. **Индекс Красоты**

Он оценивает эстетическую привлекательность и целостность пространства. В текущей модели он рассчитывается на основе трех эвристик:

  * **Цветовая гармония:** Анализирует совместимость основных цветов интерьера на основе классических цветовых схем.
  * **Стилевое единство:** Оценивает соответствие мебели заявленному стилю помещения.
  * **Пространственная эффективность:** Анализирует баланс между занятым и свободным пространством, предотвращая как «пустоту», так и «беспорядок».

## Ключевые особенности

  * **Комплексный анализ:** Расчет более 10 различных показателей, от теплопотерь в ваттах до индекса цветовой гармонии.
  * **Интеллектуальные рекомендации:** Система не только выявляет проблемы, но и запускает симуляции «что, если», чтобы предложить наиболее эффективные улучшения.
  * **Профессиональная визуализация данных:** Автоматическая генерация набора из 8 подробных и стильных диаграмм высокого разрешения для наглядного представления результатов.
  * **Модульная архитектура:** Весь код разделен на независимые модули (`thermal`, `optic`, `indexer`, `recommender` и т. д.), что обеспечивает его гибкость и масштабируемость.

## Как это работает: Техническая архитектура

Проект представляет собой конвейер обработки данных, управляемый центральным скриптом `pipeline_manager.py`.

1.  **Входные данные:** `pipeline_manager` получает словарь с полными данными о помещении.
2.  **Анализ:** Данные последовательно передаются в модули:

<!-- end list -->

  * `modules/thermal.py` и `modules/optic.py` для физических расчетов.
  * `modules/indexer.py` и `modules/beauty_indexer.py` для расчета индексов.
  * `modules/recommender.py` для генерации рекомендаций.

<!-- end list -->

3.  **Визуализация:** `visualizer/passport.py` получает все рассчитанные данные и создает набор файлов `.png`.
4.  **Вывод:** `pipeline_manager` возвращает единый, исчерпывающий словарь со всеми результатами анализа и путями к созданным изображениям.

## Как запустить и протестировать

Проект написан на Python и может быть легко запущен локально.

**1. Клонируйте репозиторий:**

```bash
git clone https://github.com/Ilya-arch13/Neighbly.ArchitectureHarmonyIndexSystem.git
cd Neighbly.ArchitectureHarmonyIndexSystem
```

**2. Установите зависимости:**

```bash
pip install -r requirements.txt
```

**3. Запустите демонстрацию:**

```bash
python examples/pipeline_demo.py
```

После запуска скрипта в консоли появится полный отчет об анализе образца комнаты, а в папке pipeline\_results будет создан набор из 8 визуализаций высокого разрешения.
